Hexo常用命令

常见命令

hexo new "postName" #新建文章

hexo new page "pageName" #新建页面

hexo generate #生成静态页面至public

hexo service #开启预览访问端口

hexo deploy #部署到guthub

hexo help #查看帮助

hexo version #查看hexo版本

组合命令

hexo s -g #生成本地预览

hexo d -g #生成并上传
